## Deskripsi Aplikasi

Aplikasi ini dibangun untuk memenuhi Ujian Tengah Semester (UTS) mata kuliah Pemrograman Seluler. Aplikasi ini mendemonstrasikan penguasaan materi:

1. **UI Responsif** (Portrait & Landscape menggunakan layout-land)
2. **Navigasi & Intent** (Mengirim data dari login ke panel)
3. **Control Flow:**

   * If-Else untuk menentukan status
   * Loop untuk generate data penilaian
